系统参数调用T_SYS_PARAM表

 

JSP页面调用

//OGNL表达式

${application.ParamSys.DEMO}

${application.ParamSys.DEMO[‘1’]}

//JSON 型参数获取

var json = ${application.ParamJson.DEMO};

 

JAVA中调用

//ACTION中获取值栈

ValueStack vs = ActionContext.getContext().getValueStack();

//系统参数调用工具对象

ParamsUtil p =new ParamsUtil(vs);

//调用DEMO类型参数 获取键值对

String rs = p.getValue("DEMO", value);

 

Oracle 中调用

select F_PARAMS('DEMO',col) from dual;
